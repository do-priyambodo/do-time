# do-time

A premium, visually stunning, and highly functional time management by Doddi Priyambodo

## 🌟 Features

*   **World Clock & Zone Comparison**: Compare time zones across 50+ popular cities and capitals with explicit GMT offsets.
*   **Alarm Clock**: Set alarms with a custom, procedurally generated chime sound (Web Audio API).
*   **Countdown Timer**: Set hours, minutes, and seconds with sound notification on completion.
*   **Stopwatch**: Millisecond precision with lap tracking.
*   **Dynamic Backgrounds**: Smooth UI gradients that change automatically based on the time of day (Sunrise, Noon, Sunset, Night).
*   **Reorderable Dashboard**: Drag and drop cards vertically to organize your focus.
*   **Focus Mode**: Collapse any section to keep your workspace clean.
*   **Local Persistence**: Your cities, alarms, and layout order are saved in your browser's local storage.

## 🛠️ Tech Stack

*   **Framework**: Next.js 16 (App Router)
*   **Styling**: Tailwind CSS
*   **Icons**: Lucide React
*   **Animations**: Framer Motion
*   **State/Audio**: React Hooks & Web Audio API

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18+ recommended)
*   npm

### Installation

1.  Navigate to the web project directory:
    ```bash
    cd web
    ```

2.  Install the dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.
